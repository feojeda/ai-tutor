## 🚀 इंस्टॉल करें

```bash
git clone https://github.com/feojeda/ai-tutor.git ~/.ai-tutor && python3 ~/.ai-tutor/install.py
```

### अपने एजेंट से इंस्टॉल करवाएं

किसी भी CLI एजेंट में यह पेस्ट करें:

```
https://github.com/feojeda/ai-tutor.git को ~/.ai-tutor में क्लोन करें और python3 ~/.ai-tutor/install.py चलाएं। फिर मुझे बताएं कि कौन से एजेंट डिटेक्ट हुए।
```

### अनइंस्टॉल करें

```bash
python3 ~/.ai-tutor/install.py --uninstall && rm -rf ~/.ai-tutor
```

इंस्टॉलर यह पता लगाता है कि आपके पास कौन से एजेंट हैं और स्किल को सही जगह पर कॉपी करता है:

```
✅ OpenCode: Installed → ~/.opencode/skills/tutor/
✅ Claude Code: Installed → ~/.claude/skills/tutor/
✅ कोर्स 'ejemplo-ml': Installed → ~/.ai-tutor/cursos/ejemplo-ml/
```

### विकल्प

```bash
python install.py --list          # डिटेक्ट किए गए एजेंट देखें
python install.py --agent opencode # केवल एक एजेंट के लिए इंस्टॉल करें
python install.py --dry-run       # बिना बदलाव के पूर्वावलोकन
python install.py --uninstall     # सभी एजेंट से हटाएं
python install.py --course name   # विशिष्ट कोर्स इंस्टॉल करें
```

## यह क्या है

मार्कडाउन में स्किल्स और कोर्स का एक सेट, जो आपके कोडिंग एजेंट में लोड होने पर उसे एक इंटरैक्टिव शिक्षक में बदल देता है। यह जानकारी नहीं उगलता — यह सवाल पूछता है, आपकी समझ की जांच करता है, और गति को आपके स्तर के अनुसार ढालता है।

दो भाग:

- **ट्यूटर स्किल** (`skills/tutor/SKILL.md`) — यह परिभाषित करता है कि कैसे पढ़ाना है: सुकराती विधि, समझ की जांच, अनुकूली गति। विषय-स्वतंत्र।
- **कोर्स** (`cursos/`) — यह परिभाषित करता है कि क्या पढ़ाना है: पाठ्यक्रम, अभ्यास, परियोजनाएं। प्रति विषय एक कोर्स।

## अपना खुद का कोर्स बनाएं

1. `cursos/ejemplo-ml/` को नए नाम से कॉपी करें
2. `curriculum.md` को अपनी सामग्री से संपादित करें
3. `ejercicios/` में अभ्यास जोड़ें
4. `python install.py --course आपका-कोर्स` चलाएं

कोर्स की संरचना:

```
cursos/आपका-कोर्स/
├── curriculum.md    ← अध्ययन योजना (सत्र, उद्देश्य, अभ्यास)
├── rules.md         ← (वैकल्पिक) विषय-विशिष्ट शैक्षणिक नियम
└── ejercicios/      ← (वैकल्पिक) विस्तृत अभ्यास
    ├── 01-intro.md
    └── 02-advanced.md
```

### अच्छे पाठ्यक्रम के नियम

1. **एक सत्र = एक अवधारणा।** एक ही सत्र में दो नए विचार न डालें।
2. **सिद्धांत के बाद अभ्यास।** हर सत्र में कम से कम एक व्यावहारिक अभ्यास होना चाहिए।
3. **समझ की जांच।** हर सत्र को एक प्रश्न के साथ समाप्त करें जो सीखने की पुष्टि करे।
4. **स्पष्ट पूर्वापेक्षाएं।** छात्र को ठीक-ठीक पता होना चाहिए कि शुरू करने से पहले क्या चाहिए।
5. **वास्तविक उदाहरण।** खिलौना उदाहरण नहीं, बल्कि वास्तविक दुनिया के डेटा और समस्याओं का उपयोग करें।

- छोटी जीत का जश्न मनाए
